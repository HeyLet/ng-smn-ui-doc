SMN UI


Introdução

Depois de bastante tempo dedicado por diversos colaboradores foi desenvolvido este módulo para Angular.

Temos como objetivo facilitar o desenvolvimento de sistemas em Angular e como foco ser utilizado internamente na empresa de desenvolvimento de softwares SMN, mas disponível também para o público através de seu repositório no GitHub e pacote NPM.

Este módulo contém componentes seguindo as diretrizes do Material Design e outras utilidades (que são usadas internamente e disponíveis para serem usadas externamente).

Neste site se encontra a documentação e exemplos do uso dos componentes e utilidades.

Desenvolvido pela SMN.

Instalação

1. Abra a pasta do seu projeto Angular através do CMD ou do GitBash e digite a seguinte linha de comando: 

            npm install ng-smn-ui --save

2. Abra o arquivo *.module.ts do seu projeto e importe o SMN UI. 

            import {{'{'}}SMNUIModule{{'}'}} from 'ng-smn-ui';

3. Ainda no arquivo *.module.ts do seu projeto e injete o SMN UI dentro do seu módulo na seção imports. 

            ...
                ],
                imports: [
                    BrowserModule,
                    SMNUIModule
                ],
                providers: [
            ...

4. Pronto. O SMN UI já está disponível para ser utilizado no seu projeto.
