SMN UI


Introdução

Depois de bastante tempo dedicado por diversos colaboradores foi desenvolvido este módulo para Angular.

Temos como objetivo facilitar o desenvolvimento de sistemas em Angular e como foco ser utilizado internamente na empresa de desenvolvimento de softwares SMN, mas disponível também para o público através de seu repositório no GitHub e pacote NPM.

Este módulo contém componentes seguindo as diretrizes do Material Design e outras utilidades (que são usadas internamente e disponíveis para serem usadas externamente).

Neste site se encontra a documentação e exemplos do uso dos componentes e utilidades.

Desenvolvido pela SMN.

Introduction(Em construção)

After a long time of dedication from several collaborators it was developed this module for Angular.

We have as objective facilitate the development of systems with Angular and focusing on being used internally on the company of software development SMN, but it is also available for the public on its repository at GitHub and on the NPM package.

This module contains components following the Material Design guidelines and other utilities (that are used internally and also available for being used externally).

On this website you can find the documentation and examples of the use of the components and utilities.

Developed by SMN.

Installation

1. Open the directory of your project Angular through CMD or GitBash and type the following command line: 

    npm install ng-smn-ui --save

2. Open the file *.module.ts of your project and import the SMN UI. 

    import {{'{'}}SMNUIModule{{'}'}} from 'ng-smn-ui';

3. Still on the file *.module.ts of your project, inject the SMN UI inside your module on the section imports. 

    ...
        ],
        imports: [
            BrowserModule,
            SMNUIModule
        ],
        providers: [
    ...

4. Done. The SMN UI is already available to be used on your project.
