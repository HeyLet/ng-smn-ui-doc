Atributos 


    modelValue: Pega o valor selecionado.

    list: Lista de opções para o select (pode ser um array de strings ou um array de objetos).

    loading: Utilizado para controlar a barra de carregamento do select.

    select: Dispara um evento após a seleção de um item.

    loadMore: Evento disparado quando o scroll do autocomplete chega perto do final da barra de rolagem.

    theme-class: Usado para definir o tema.

    primary*: Utilizado para definir o item principal do autocomplete.

    secondary*: Utilizado para definir o item secundario do autocomplete.

    *list deve ser um array de objetos

JS

search() {
clearInterval(this.timing);
this.loading = true;
this.timing = setTimeout(() => {
this.loading = false;
if (!this.searchTeams) {
this.teamsFiltrados = this.teams;
return;
}
this.teamsFiltrados = this.teams.filter(item => {
return (item.indexOf(this.searchTeams) !== -1);
});
}, 300);
}
finish(item) {
console.log(item); // Mostra o item selecionado
}
