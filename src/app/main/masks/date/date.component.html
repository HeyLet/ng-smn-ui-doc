Date Mask

Adiciona uma máscara de data ao campo, e converta a saida para uma string de data.

Exemplo

Aceita [minDate] e [maxDate] como parâmetro do tipo data, ele estipula uma data mínima e máxima para que o campo seja válido.

[dateFormat] informa qual o formado de data a ser adotado, padrão 'dd/MM/yyyy'
