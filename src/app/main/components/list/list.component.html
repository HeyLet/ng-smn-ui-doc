Lists

Listas são uma organização vertical continua de textos ou imagens.

Dinamica:

É necessario informar a fonte de dados da lista através do atributo "list", e também um modelo de item, através do atributo "item-model"

Mista:

É possivel criar uma lista com itens estáticos e dinamicos.
