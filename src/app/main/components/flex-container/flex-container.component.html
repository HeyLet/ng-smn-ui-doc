Flex Container

Flex container é uma ferramenta de layout. Utiliza o modelo de flex-box para organizar um conteudo.

Exemplo

Um flex container toma toda a largura disponivel, se houverem vários containers lado a lado o espaço é distribuido entre eles.

A classe break-on-s** informa que o container deve tomar todo o espaço possível a partir de uma largura da pagina

Redimencione a pagina e veja o que acontece

Os valores suportados são break-on-s1600, break-on-s1440, break-on-s1280, break-on-s960, break-on-s840, break-on-s600 e break-on-s480
